CONFIGURASI SNTP CLIENT MIKROTIK

Post November 29, 2021 by Miklombok |
Sntp-client

Mikrotik Sebagai NTP Client Pada package system RouterOS Mikrotik sudah terdapat fitur SNTP (Simple Network Time Protocol) Client yang bisa digunakan
untuk memfungsikan Router sebagai NTP Client. Saat menyala, Router akan otomatis melakukan sinkronisasi waktu terhadap NTP
Server yang ditunjuk sehingga pengaturan waktu akan tetap update.

Langkah - langkah konfigurasi sntp client mikrotik: :
1. Klik menu System -> SNTP Client
- Centang Enable
2. isi Primary NTP Server: 0.id.pool.ntp.org
3. isi Secondary NTP Server: 1.id.pool.ntp.org
dan jangan lupa menggunakan mode=unicast..
