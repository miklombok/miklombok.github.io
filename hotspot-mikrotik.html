SETTING MIKROTIK SYSTEM VOUCHER

Post November 30, 2021 by Miklombok |
Mikrotik hotspot

Hotspot di Mikrotik adalah sebuah system untuk memberikan fitur autentikasi pada user yang akan menggunakan jaringan.
Jadi untuk bisa akses ke jaringan, client diharuskan memasukkan username dan password pada login page disediakan. Sumber: http://mikrotik.co.id/.

Langkah Langkah Setting Hotspot Mikrotik :
1. Bridge Port
- Klik Bridge -> +
Name: BridgeLAN dan OK.
- Klik ke Tab Port -> +
Pilih interface LAN, disini port LAN memakai ether2,ether3,dan ether4. port ini yang akan disambungkan ke wifi hotspot.

2. DHCP Client Mikrotik
- Klik IP -> DHCP Client ->+
Pilih Interface WAN disini menggunakan ether1 pada mikrotik dan tekan OK agar mendapat IP dari router ISP dan status sampai status bound.

3. Address List Mikrotik
Klik IP -> Address -> Address List ->+
Address: 192.168.222.1/24
Network: 192.168.222.0
Interface: BridgeLAN (interface bridge yang sudah dibuat )

4. DNS Settings Mikrotik
Servers: 8.8.8.8 (DNS google) atau bisa menggunakan cloudflare 1.1.1.1.
Dynamic Servers: (terisi otomatis, DNS dari ISP modem).
Centang Allow Remote Requests dan tekan OK.

5. Firewall NAT Mikrotik
Klik IP -> Firewall -> NAT -> +
Chain: srcnat
Out Interface List: ether1
Klik Tab Action
Action: masquerade setalah itu apply - ok

Setting Mikrotik dasar sudah selesai. Coba lakukan pengetesan akses internet dan test speed atau ping dari mikrotik ke 8.8.8.8. Klik New Terminal ketik ping 8.8.8.8.
6. Hotspot Mikrotik
Klik IP -> Hotspot ->Hotspot Setup -> +.
Hotspot interface : BridgeLAN
Local address: otomatis terisi
Address pool: Otomatis Terisi
select certificate : none
ip address of SMTP server : 0.0.0.0
Dns server : otomatis terisi
Dns name : balehotspot.net ( sesuaikan dengan keinginan anda sendiri )
username dan password silahkan di sisi sendiri kemudian ok
