Install Mikhmon Android

Post November 28, 2021 by Miklombok |
Mikhmon

1. Download aplikasi termux di Playstore.
2. Buka aplikasi termuxnya dan ikutin langkah di bawah ini
3. Install paket yang dibutuhkan untuk mikhmon.
Keterangan :
- apt update && apt upgrade -y (update dan upgrade termux).
- apt install curl (install curl).
- curl -o install-mikhmon https://laksa19.github.io/install-mikhmon.txt (install curl di server mikhmon).
- chmod +x install-mikhmon (memberi hak akses agar bisa diexsekusi).
- ./install-mikhmon (install mikhmon).
- Buka url http://0.0.0.0:8080 di browser Anda.

Apabila semua paket mikhmon sudah di install di termux kemudian instalasi tida berjalan pada saat melakukan Perintah

./install-mikhmon

silahkan ketikk printah :
1. apt update
2. apt upgrade
jika berhasil silahkan lakukan instal paket ./install-mikhmon
